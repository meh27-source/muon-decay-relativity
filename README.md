# Muon Decay, Special Relativity and Linear Algebra

## About This Project

This project started several months ago from a combination of interests that I had been developing at the same time.

I was learning Python and becoming familiar with libraries such as NumPy and Matplotlib. At the same time, I was fascinated by Einstein's Theory of Special Relativity and the famous example of muons reaching Earth's surface despite their extremely short lifetime.

What began as a simple idea to visualize exponential decay gradually grew into a much larger project. As I continued learning mathematics and physics, I kept returning to the project and expanding it with new concepts.

Eventually, topics such as Monte Carlo simulation, probability distributions, matrix operations and Lorentz transformations became part of the journey.

Rather than being built all at once, this repository reflects how my understanding developed step by step.

---

## The Question That Started Everything

Muons have an average lifetime of only:

**2.2 microseconds**

At first, this seemed surprising.

Many muons are produced high in Earth's atmosphere, yet they are still detected at ground level. Based on their short lifetime, it appeared that most of them should decay long before completing the journey.

Understanding why this happens became the central motivation for this project.

---

## Project Structure

### 01 — Exponential Decay

The project begins with the standard exponential decay model.

In this section, I explored how the number of surviving muons decreases over time and visualized the decay curve using Python.

**Topics:**

* Exponential decay
* Scientific computing
* Data visualization

---

### 02 — Special Relativity

The second section introduces Einstein's Special Theory of Relativity.

By applying time dilation, I investigated how the observed lifetime of fast-moving muons changes and why relativistic effects are necessary to explain real observations.

**Topics:**

* Time dilation
* Lorentz factor
* Relativistic decay

---

### 03 — Monte Carlo Simulation

Instead of only studying average behaviour, I began simulating individual muons.

Each muon was assigned a random lifetime drawn from an exponential distribution, allowing me to explore the statistical side of particle decay.

**Topics:**

* Random sampling
* Probability distributions
* Monte Carlo methods

---

### 04 — Muon Survival Simulation

This section combines decay, relativity and probability.

Using simulated lifetimes, I investigated how many muons survive the journey from the upper atmosphere to Earth's surface and compared the results with and without relativistic effects.

**Topics:**

* Survival probability
* Simulation
* Experimental interpretation

---

### 05 — Linear Algebra Formulation

As I learned more mathematics, I wanted to connect the project to linear algebra.

Muon decay was reformulated using state vectors and transition matrices. I also explored matrix powers and Lorentz transformation matrices to show how linear algebra can be used to describe physical systems.

**Topics:**

* Vectors
* Matrices
* Matrix multiplication
* Matrix powers
* Lorentz transformations
